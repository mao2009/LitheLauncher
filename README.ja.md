# LitheLauncher

LitheLauncherは、Python (PySide6) で開発された「爆速」ゲームランチャーアプリケーションです。
5,000件を超える膨大なゲームライブラリでも、**0.12秒**という驚異的な速度で起動し、非同期画像デコード技術により、大量のカバーアートを表示しながらも一切のカクつきがない滑らかなスクロール体験を提供します。

## 主要機能

-   **爆速起動 & 非同期ロード**: 5,000件以上のゲームデータがあっても、UIをブロックせずに瞬時に起動し、バックグラウンドで逐次データを読み込みます。
-   **高度な非同期画像処理**: 専用のスレッドプール (`ImagePool`) を使用し、画像のデコードとスケーリングをバックグラウンドで実行。スクロール時のUIラグを完全に排除しています。
-   **仮想化グリッド表示**: `WidgetPool` によるウィジェットの再利用と仮想化技術により、メモリ使用量を抑えつつ膨大なリストを高速に描画します。
-   **起動前/起動後コマンド**: ゲームの起動前後にカスタムコマンドを実行できます。MODの適用や設定のカスタマイズに便利です。
-   **スマート・セーブデータ同期**: ローカルとリモートのタイムスタンプを比較し、最新のデータを自動的に同期。不整合発生時の競合解決UIも備えています。
-   **コードファーストDBマイグレーション**: スキーマ変更時の自動バックアップとリカバリプロトコルにより、データ整合性を強力に保護します。
-   **多言語対応 (i18n)**: 日本語と英語をフルサポート。システムロケールの自動検出およびリアルタイム切り替えが可能です。
-   **テスト自動化**: `pytest-qt` および `Hypothesis` を用いた、GUIとロジックの網羅的なテスト自動化。

## ターゲットユーザー

LitheLauncherは、特に数千本規模の膨大なゲームライブラリを所有し、既存のランチャーの重さに不満を感じているパワーユーザーに最適です。

## インストール

### 前提条件

-   Python 3.x (推奨 3.9以上)
-   pip (Pythonパッケージインストーラー)
-   Git (リポジトリのクローン用)

### 手順

1.  **リポジトリのクローン**:
    ```bash
    git clone https://github.com/your-username/lithelauncher.git
    cd lithelauncher
    ```
2.  **仮想環境の作成**:
    ```bash
    python -m venv venv
    ```
3.  **仮想環境のアクティブ化**:
    -   **Windows**:
        ```bash
        .\venv\Scripts\activate
        ```
    -   **macOS/Linux**:
        ```bash
        source venv/bin/activate
        ```
4.  **依存関係のインストール**:
    ```bash
    pip install -r requirements.txt
    ```

## 利用方法

### アプリケーションの起動

インストール後、仮想環境をアクティブ化し、以下を実行します。

```bash
python main.py
```

### ゲームの追加

1.  「ゲームを追加」ボタンをクリックします。
2.  実行可能ファイルパスを含むゲームの詳細を入力します。
3.  オプションで、起動前/起動後コマンドおよびセーブデータ同期設定を行います。
4.  ゲームを保存します。

### 起動前/起動後コマンド

ゲーム起動前（例: MODを有効にする）やゲーム終了後（例: 特定のファイルをバックアップする）に実行するコマンドを設定できます。

1.  ゲームの詳細を編集します。
2.  「起動前コマンド」および「起動後コマンド」フィールドに目的のコマンドを入力します。
3.  コマンドはシステムシェルを使用して実行されます。ご使用のOSで有効であることを確認してください。

### セーブデータ同期

ゲームの詳細でセーブデータ同期を有効にすると、セーブファイルがリモートの場所へ自動的にアップロード/ダウンロードされます。

1.  「セーブデータ同期」を有効にします。
2.  ローカルの「セーブフォルダ」と「リモート同期パス」を指定します。
3.  ゲーム起動前にセーブデータがダウンロードされ、終了後にアップロードされます。
