# LitheLauncher

LitheLauncherは、Pythonで開発された「爆速」ゲームランチャーアプリケーションです。
5,000件超のライブラリでも **0.12秒** で起動し、高度な非同期処理により一切のカクつきがない操作感を実現しています。

## 主要機能

-   **爆速起動 & 非同期ロード**: 膨大なデータがあってもUIをブロックせず瞬時に起動します。
-   **非同期画像デコード**: `ImagePool` によるバックグラウンド処理で、スクロール時のラグを完全に排除。
-   **起動前/起動後コマンド**: MOD適用や設定変更などのカスタムコマンドを自由自在に実行。
-   **スマート同期**: セーブデータの自動同期と競合解決。
-   **堅牢なデータ管理**: 自動バックアップ付きのマイグレーションエンジン。
-   **フル・テスト自動化**: `pytest-qt` 等による高品質なコードベース。

## ターゲットユーザー

膨大なゲームライブラリを所有し、軽量で高速な管理環境を求めるパワーユーザー。
